Wireless Network

The wireless network on campus is a service administrated by the University of Maryland Department of Information Technology. Instructions to get connected can be found here.

UMD Directory ID holders

Use the eduroam network with your @umd.edu email address and directory password.

Non-UMD Directory ID holders (Guest Access)

Individual access

Sign up for and connect to the umd-guest network.

Bulk access

Contact UMIACS Staff.

Wired Network

UMIACS provides wired network access to its offices, labs, and centers. These ports are typically identified by having a yellow port (Iribe) or orange port (AVW), and a 5-character identification string.

Wired Connection for personal machine

To get a wired port enabled for a personal laptop or desktop, please send mail to staff@umiacs.umd.edu noting the room number and identification number of the port.
