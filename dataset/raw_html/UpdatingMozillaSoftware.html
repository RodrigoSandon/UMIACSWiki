Like most applications, it is important to keep your browser and email clients up-to-date to combat security flaws and gain new functionality. The instructions on this page should help you update Mozilla Firefox and Thunderbird.

You can also find installers and documentation for the latest versions of Firefox and Thunderbird here:

  • Firefox: http://www.getfirefox.com
  • Thunderbird: http://www.getthunderbird.com

Determining the Installed Version

You can find out what version of Firefox or Thunderbird you have installed on you computer by opening up the application and choosing:

  • "Help > About Mozilla [application]" from the menu bar on Windows
  • "Firefox > About [application]" from the menu bar on macOS

where [application] is either "Firefox" or "Thunderbird".

Updating Mozilla Firefox (Windows & Mac)

  • By default Mozilla Firefox will automatically update by itself. If it is currently disabled it can be enable with the following steps:
  1. First navigate to options (Windows) or preferences (Mac).
  2. Once in the options (Windows) or preferences (Mac) you navigate to the Firefox Updates section
    • Once in the Firefox Updates section you will see an option to “Automatically install updates”. If this is not already enabled it is recommended to enable this.

  4. Click “Restart to Update” button (Windows and Mac)

Updating Mozilla Thunderbird (Windows & Mac)

  • By default Mozilla Thunderbird will automatically update by itself. If it is currently disabled it can be enable with the following steps.
  1. First navigate to options->options(Windows) or preferences(Mac).
  2. Once in the options(Windows) or preferences(Mac) you navigate to the advanced tab.
  3. Once on the advanced Tab you navigate to Updates.
    • Once in the updates tab you will see an option to “Automatically install updates”. If this is not already enabled it is recommended to enable this.

  1. First navigate to help->about Thunderbird (Windows) and about Thunderbird (Mac).
  2. Click the check for updates button (Windows and Mac).
  3. Click “Update & Restart” button (Windows) and “Restart to Update” button (Mac).

Troubleshooting

If for some reason Firefox or Thunderbird refuses to update using the methods described above, please stop by the HelpDesk and have a technician assist you.
